# Welcome to Checkpoint

Checkpoint is a modern version control system built for media-heavy workflows: games, film, product design, architecture, and more.

## Why Checkpoint?

Traditional version control systems like Git were designed for source code. Checkpoint is purpose-built for projects with large binary files — textures, 3D models, video, CAD drawings — where performance and workflow matter.

### Key Features

- **Fast Checkouts** — Efficiently handle repositories with large files without bloating history
- **File Locking** — Prevent conflicts on binary files that can't be merged
- **Branching & Merging** — Familiar workflows adapted for media-heavy projects
- **Massive File Support** — No artificial file size limits; large binaries are first-class citizens
